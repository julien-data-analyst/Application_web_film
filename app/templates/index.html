Présentation de l'outil décisionnel

Expliquation technique


CinéAnalytics est un outil décisionnel qui permet aux professionnels de l'industrie, chercheurs, et autres passionnés de cinéma, de plonger efficacement et intuitivement dans le monde des données cinématographiques. Chaque utilisateur peut visualiser diverses réalisations graphiques disponible ci-dessous, ou encore utiliser notre moteur de recherche Les données sont enregistrés dans une base SQLite ; réparties en 11 tableaux (table) dont 4 étant des tableaux d'associations. Le modèle logique des données sépare les variables GENRE, COMPANY, LANGUAGE, COLLECTION, ACTOR, et DIRECTOR du tableau principal.

    Langages utilisés :

  • Python
  • HTML
  • CSS
  • JavaScript
